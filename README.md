## 项目简介

EPG（Electronic Program Guide）电子节目单生成工具，用于从多个数据源抓取电视节目单。

---

## 🚀 功能特性

| 功能                         | 描述                             |
|------------------------------|----------------------------------|
| 多数据源支持                 | 支持从多个数据源抓取节目单        |
| 自动合并、去重               | 节目数据自动合并去重              |
| 频道标准化                   | 支持频道名称标准化和映射          |
| XMLTV格式                    | 生成标准 XMLTV 格式输出           |
| 配置文件自定义               | 支持配置文件自定义                |
| GZIP输出                     | 支持 GZIP 压缩输出                |
| 日志记录                     | 支持日志记录                      |
| 模糊匹配                     | 支持模糊匹配频道                  |
| 成功率阈值                   | 支持成功率阈值配置                |

---

## 📡 支持的数据源

- 央视节目单
- 卫视频道节目单
- 部分地方台节目单

## 🚀 部署说明

1. 克隆仓库到本地  
   ```bash
   git clone https://github.com/37289551/SH.git
   ```
2. 安装依赖  
   ```bash
   pip install -r requirements.txt
   ```
3. 在 GitHub 仓库中设置 Secrets  
4. 启用 GitHub Actions 工作流  
5. 工作流会自动定时运行，生成并更新 EPG 数据
